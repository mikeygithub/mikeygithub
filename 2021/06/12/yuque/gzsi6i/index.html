总结篇-实习记录

初出茅庐

三月九号开始独自一人踏上征程，飞了一千多公里来到了这个陌生的城市－－杭州。不得不说航班延误是真的让人烦躁，原本正常八点多就可以到达萧山机场，硬是到了十点多钟才到，导致到余杭后已经快十二点了。打车根本打不到，幸运的是赶上了最后一趟公交车到了一个宾馆住了下来。

租房相关

租房需要考虑的因素主要有通勤、价格、周围环境、隔音效果、合同条款、设备是否齐全，特别是合同条款需要注意，需要了解清楚如果自己中途需要退房如何处理，押金怎么办。杭州这边的城中村基本全部拆完，普遍都是小区出租，租金都比较贵一些，而且都是那种隔断间，隔音效果极差，如果靠近公路尽量租高楼层，这样噪音不那么严重。还有一个是看你自己的选择，有的人比较喜欢合租闲暇之时可以聊聊天、打打游戏，反正我是不喜欢合租哈哈。

在这里要感谢一下夏导，在得知我到杭州后还抽空出来陪我去房租，给我介绍的地方十分不错，不过我感觉略微远了点，就没有租下。在我租下房子后还带我去阿里访客中心溜达了一遍。

要有自己的判断力，不要被那些房东阿姨的花言巧语给忽悠了，不要急着租下来，房子肯定是会有的，大可不必担心被别人先租。

工作相关

刚刚入职那几天最不适宜的就是公司的电脑是 MAC，让我这个老 Ubuntu 很是不适应，就像个傻子一样瞎捣鼓。一般都是将开发的环境配置好后导师开始将解项目，大概对我们组负责的产品进行大概的介绍，确实产品的功能非常多，链路也是比较长，理解起来也比较费劲。
​

入职第一周周末部门就进行 POC 演练，原本主管的想让我们观摩学习谁知道我被分配的那个小组并不知道我是个实习生，直接对我分配了任务，好家伙幸亏有点底子。先是对一些服务器环境的配置，安装 NGINX，部分微服务应用，普罗米修斯监控+格兰芬那大盘，期间遇到很多坑，请教了一些同学才得以解决，两天周末一天干到十点多、一天干到一点多无疑是让我有点慌了，要是经常是这个强度这谁顶得住。后期主要是那些大佬们对搭建起来的系统进行调优，提高 TPS、降低 RT。
​

通过 POC 演练还是能学习到很多东西的，就比如说多学会了几个 Linux 的相关命令，系统调优的方法等等。

​

第二次导师带我们三个实习生定了个会议室，对系统的各个功能进行演示一遍。开始制定一些每周的要求，记录都干了什么事情，需要一个月以后进行一次答辩，考察我们的掌握情况（后面考察不成了，导师离职了）。开完会回来主管突然分配了个任务，叫我们解析 3G 多的 XML 文件，因为文件太大只能用 SAX 来进行解析了，花了我好几天时间。其中里面有两个数据结构及其复杂，根本不知道其数据量的大小，调试了好久，逻辑没控制好导致一直写了一百多 G 数据进入 MySQL 中，现在心想还真危险。所以以后当遇到与实际逻辑不相符时一定要敢于怀疑、善于怀疑才能避免一些重大事故的发生。
​

开始的几周就马上投入了项目的修改 BUG 中去了，问题全部靠自己悟，导师任务繁重，去问他问题显得十分不耐烦，后面也不敢问他了，直接说ｘｘｘ版本的服务标示没有，叫我加一下，对于我这个还没很熟悉的菜鸟来说显得很吃力，花费了两天时间，但是后面加成功了，在本地进行测试也有了，因为是版本不同我没能连接到生产数据库，使用的是当前新版本数据库。提到他们生产环境后就没有出现。后来我就直接和导师说我排查不出原因，最后他看了一下发现生产环境数据库没有刷入那个字段导致的，加上去就可以了。这个最终也导致我在周会上被点名了一次。
​

在遇到问题要冷静分析问题发生的原因，制定好相关排查计划，不要一股脑就上去直接开干，很容易跳进坑。

组长是一位技术专家，技术能力极强，为人非常亲和，出差回来看到我们就跟我们说有问题就要多问问他们，而且在周会我被点名的时候，他主动站出来为我说话，挺感激他的。但是一周之和就离职了，希望以后江湖再见吧。
​

接着陆陆续续参加相关项目的开展工作、showcase、codereview、项目需求评审、用例评审、技术评审的工作。确实在这样的环境中能学习到挺多东西。一个多月以后开始写项目的需求了，另外两位实习生开始陆续回校了，两位同事的离职，其中有一个就是导师。项目的开发人员就剩下一个实习生＋一个入职六个月的开发了，还好他们是在当前版本的功能开发完的情况下才离职的，接着另一个同事开始出差半个月，在这半个月里就剩下我一个开发，测试开始进行测试提出的 BUG 都是由我来进行 FIX，压力肯定是有，但是也能加快我的成长。
​

聊一聊导师吧，导师技术能力也是没得说，之前都是在大厂待的，看起来非常的年轻，像是二十刚出头的小伙子一样。很感谢他在这两个月里的教导，可能是主管离职后导致工作都压到他的身上，想对我们展开考察都没有时间。在他要离职之前还请了我们喝奶茶，同时点出来我的一些不足，表达能力这一块还有待提升，很幸运能遇到这样一位导师，受益匪浅。
​

在技术方面公司项目使用到的很多技术自己之前都没有深入学习、甚至还有没听说过的一些中间件，这极大提升了我的学习欲望，三个月的摸爬滚打，对这些技术也都有了进一步的理解，想达到更高的高度还是需要在实际项目中进行实践探索。
​
